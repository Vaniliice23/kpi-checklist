# КПИ · Чек-лист отделов

Мобильное приложение для управления мотивацией старших продавцов по отделам. Работает на Android и iOS, синхронизируется с Google Таблицей через Apps Script.

## Быстрый старт

### 1. Создайте репозиторий на GitHub

1. Откройте [github.com/new](https://github.com/new)
2. **Имя репозитория**: `kpi-checklist` (или любое другое)
3. **Видимость**: Public
4. Галочка **"Add a README file"** — не требуется
5. **Create repository**

### 2. Загрузите файлы приложения

Есть два способа:

#### Способ A: Через веб-интерфейс (простейший)
1. На странице репозитория нажмите **Add file → Upload files**
2. Перетащите туда эти 4 файла:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - (опционально) `README.md` — если хотите виджет инструкции в репо
3. **Commit changes** (скроллите вниз)

#### Способ B: Через командную строку (для опытных)
```bash
git clone https://github.com/ВАШ_ПОЛЬЗОВАТЕЛЬ/kpi-checklist.git
cd kpi-checklist
# скопируйте 4 файла в эту папку
git add .
git commit -m "Initial commit: КПИ приложение"
git push
```

### 3. Включите GitHub Pages

1. На странице репозитория → **Settings**
2. Слева: **Pages**
3. **Source**: выберите **Deploy from a branch**
4. **Branch**: `main` и папка `/ (root)`
5. **Save**

Через 1–2 минуты появится зелёная ссылка: **Your site is live at `https://ВАШ_ПОЛЬЗОВАТЕЛЬ.github.io/kpi-checklist`**

### 4. Откройте приложение на телефоне

1. Скопируйте эту ссылку → откройте в браузере телефона (Chrome на Android или Safari на iOS)
2. При первом открытии приложение попросит **URL Google Apps Script**
3. Вставьте URL вашего Apps Script (заканчивается на `/exec`)
   - Пример: `https://script.google.com/macros/s/AKfycbx1HaUB3L7I.../exec`
4. Нажмите **Сохранить**

### 5. Установите как приложение

#### На Android (Chrome)
1. Откройте ссылку приложения
2. Меню ⋮ (три точки вверху справа)
3. Прокрутите вниз
4. **Установить приложение** (или **Install app**)
5. **Установить**

Иконка появится на главном экране и будет открываться как полноценное приложение.

#### На iOS (Safari)
1. Откройте ссылку приложения
2. Нажмите иконку **Поделиться** (из центра)
3. Прокрутите влево
4. **На экран Домой** (или **Add to Home Screen**)
5. **Добавить**

Иконка появится на главном экране — открывается как приложение.

## Что дальше

- **Отметки сохраняются мгновенно** в вашу Google Таблицу (через Apps Script)
- **Работает офлайн** — отметки кэшируются локально и синхронизируются при появлении сети
- **Все данные в Google Таблице** — исходная структура «Лог», «Отчёт» и т.д. сохраняется

## Техподдержка

Если приложение:
- **Не грузится**: проверьте интернет и URL Apps Script (должен заканчиваться на `/exec`)
- **Не сохраняет отметки**: убедитесь, что URL правильный, и что таблица доступна
- **Медленно синхронизируется**: это нормально для первого открытия или офлайн-работы — дождитесь зелёного уведомления «Сохранено»

Если нужны изменения в приложении (цвета, тексты, функции) — напишите, помогу отредактировать исходники.
